SOBRE O PROJETO

SOBRE O SITE

Este site é um projeto de estudo que consome a Demon Slayer API para exibir informações sobre personagens, onis e estilos de respiração do universo de Kimetsu no Yaiba.

CRIADOR

Projeto desenvolvido por Yuri Silva como parte da matéria de Front-End do Senai.

TECNOLOGIAS

Utiliza HTML5, CSS3 e JavaScript puro para realizar as requisições (Fetch API) e manipular o DOM dinamicamente.
